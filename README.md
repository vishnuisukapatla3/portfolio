# Personal Portfolio - Vishnu Isukapatla

A modern, responsive, and professional personal portfolio website built with **React**, **CSS3**, and **Phosphor Icons**. This project showcases my skills as a Java Full Stack Developer, highlighting my projects, experience, and education.

## 🚀 Features

- **Professional Banner Layout**: A high-impact introduction section with a dynamic architectural banner.
- **Responsive Web Design**: Fully optimized for all screen sizes, from mobile to ultra-wide desktops.
- **Interactive Tech Stack**: A visual representation of my core competencies and tools.
- **Project Detail Pages**: In-depth look at key projects, including tech stacks, challenges, and learnings.
- **Experience Timeline**: A clean, structured overview of my professional journey and achievements.
- **Contact Integration**: Fully functional contact form with automated handling.
- **Dark/Light Mode**: Toggle between sleek dark and minimalist light themes.
- **Ambient Music Player**: A subtle audio visualizer for a unique browsing experience.

## 🛠️ Technology Stack

- **Frontend**: React, JavaScript (ES6+), Vanilla CSS
- **Routing**: React Router DOM
- **Icons**: Phosphor Icons
- **Animation**: CSS Keyframes & Framer-Motion (optional integration)
- **Deployment**: Optimized for hosting on platforms like Vercel or Netlify.

## 📁 Project Structure

```text
src/
├── components/
│   ├── layout/      # Navbar, Sidebar, Footer, ScrollToTop
│   └── ui/          # Reusable UI components (MusicPlayer, etc.)
├── pages/           # Page-level components (Home, Projects, Skills, etc.)
├── data/            # Static project and skill data
├── styles/          # Global styles and variables
└── App.jsx          # Main application routing and layout
```

## 📦 Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/vishnuisukapatla3/portfolio.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run locally**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
